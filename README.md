# LyndsHub
## Un hub hecho en Python para documentar versiones. Perfecto para añadir como aplicación en tu sistema operativo.

Todo el contenido, colores, fuentes y textos se cargan desde DATA_ROOT (una variable en el código).
Lo unico que tienes que cambiar en este archivo es el DATA_ROOT.
El DATA_ROOT por defecto apunta al directorio en el que está el script.

Carpetas de configuración
```
En DATA_ROOT/pages/ están los json con la configuración de las páginas (pages).
En DATA_ROOT/assets es donde tienes que poner las imágenes de las redes sociales como github.png o discord.png porque no las añadí para evitar conflictos de derechos de autor. Puedes ejecutar el programa sin las imágenes sin problema.
```
Archivos esenciales
```
DATA_ROOT/fonts.json no te recomiendo cambiarlo.
DATA_ROOT/navigation.json tiene el orden en el que se mostrarán las páginas.
DATA_ROOT/config.json tiene configuraciones que las de arriba te recomiendo cambiar para personalizarlo.
DATA_ROOT/icon.png es el icono de LyndsHub, el que se mostrará en la ventana.
```


Ve a la línea 490, verás una pestaña base para que modifiques y te inpires, es una copia del doc.
En la línea 250 verás una función en la que tienes que especificar las páginas nuevas que pongas.

Hice este código para añadirlo como aplicación en mi sistema operativo LyndsOS.

Por cierto, todas las configuraciones base (para que las modifiques) tienen humor + real, asi que te puedes reir un poco.
